# TLex Lua Scripts

This repo contains scripts created to accomplish tasks with specific TLex dictionary databases.

## Getting Started

If you see this README, congrats, you probably already have access to this repo. `git clone`
Since this repo is a collection of scripts, it is organized with language specific scripts in their respective folders.  As scripts are no longer used, or are replaced form normal use, they are moved their respective Archived folder.

### Prerequisites

TLex 2020+

## App Usage

For Most scripts, Open the dictionary database in TLex, go to Tools->Execute Lua Script, select your script and it runs.  Keyboard shortcuts may be installed under Tools->Preferences.
